## 🚀 Quick Deploy Tutorial (2026 Edition)

Follow these steps to bring **XADON AI** online in minutes!

### Step 1: Choose & Setup a Free/Paid Bot Hosting Panel
Recommended (easy & reliable):
- [client.spaceify.eu](https://client.spaceify.eu) ← We'll use this  
- [bot-hosting.net](https://bot-hosting.net)

### Step 2: Create Your Server on Spaceify Panel
1. Go to → [client.spaceify.eu](https://client.spaceify.eu)  
2. Register → Verify email → Login  
3. Menu (☰) → **Create Server**  
   - **Name**: XADON-AI-Bot (or anything cool)  
   - **Software**: Node.js  
   - **Location**: Germany (fastest)  
   - **Storage**: Slide to **maximum**  
   - Click **Create Server**  

Your server is now live! 🎉

### Step 3: Upload & Launch XADON AI
1. Back to panel(panel.spaceify.eu)Login → Click your server name  
2. Go to **Files** tab  
3. **Upload** the XADON AI .zip file (get it from the official WA group or source)  
4. Right-click the zip → **Unarchive**  
5. Delete the .zip file  
6. Select all files in the folder → **Move** → `../` (root directory)  
7. Go to **Console** tab → Click **Start**  

### Step 4: Pair Your WhatsApp
- Watch the console – it will show a **QR code** or **pairing code**  
- Open WhatsApp → Linked Devices → Link with QR / code  
- Done! Your **XADON AI** is now live and defending chats! 🛡️🤖

> **Pro Tip**: Use `npm install` first if console shows missing modules (rare).
